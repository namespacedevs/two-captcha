
### two-captcha wrapper

<p align="center">
  <a href="https://www.npmjs.com/package/two-captcha"><img alt="NPM version" src="https://badge.fury.io/js/two-captcha.svg"></a>
</p>

Installation:
```js
npm i two-captcha
```
Usage
Set up your api key:
```js

var TwoCaptcha = require('two-captcha')
var Captcha = new TwoCaptcha('your-api-key')

```
